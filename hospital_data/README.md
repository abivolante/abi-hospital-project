# Hospital Data Sources

This folder will store data collected from publicly available UK hospital datasets. Below is a catalogue of all known public sources, organised by category.

---

## Central Portals

| Dataset | Description | Publisher | Link |
|---|---|---|---|
| NHS Digital Data Hub | Central platform for NHS data collections and statistical publications | NHS England | [link](https://digital.nhs.uk/data) |
| NHSBSA Open Data Portal | Free public access to NHS healthcare datasets | NHS Business Services Authority | [link](https://opendata.nhsbsa.net/) |
| Organisation Data Service (ODS) | Directory of NHS trusts and sites with CSV downloads and API | NHS England | [link](https://www.odsdatasearchandexport.nhs.uk/) |
| UK Government Open Data Portal | Health and social care datasets from central government | data.gov.uk | [link](https://www.data.gov.uk/) |
| ONS Health and Social Care Data | Published data on NHS, healthcare system, and patient experiences | Office for National Statistics | [link](https://www.ons.gov.uk/peoplepopulationandcommunity/healthandsocialcare/healthcaresystem/datalist?filter=datasets) |

## Accessing the Data

Most datasets are available as direct Excel/CSV downloads from the links above. Some detailed datasets (particularly HES) require an application through the [Data Access Request Service (DARS)](https://digital.nhs.uk/services/data-access-request-service-dars).
