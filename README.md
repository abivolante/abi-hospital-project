# UK Hospital Data Project

Hi, I'm **Abigail Volante**, a Mathematics student at Lancaster University. I've been passionate about numbers for as long as I can remember, maths has been a big part of my life since childhood, and studying it at university has only deepened that love.

After completing a statistics and probability course, I wanted to take that interest further and try building something hands-on with real data. That's what led me here.

## What is this project?

This is a personal learning project where I'm exploring modern data tools, specifically **dbt** (data build tool) and **Google BigQuery**. Both of my parents work in a hospital, so the healthcare world has always felt close to home. That inspired me to focus this project on UK hospitals: collecting data across different hospitals into a single, unified location, and then seeing what patterns and inferences I can draw from it.

The goal isn't just the analysis itself, it's the process of learning how to wrangle, model, and make sense of data using industry tools.

## What I'm learning

- **dbt** — for transforming and modelling data in a structured, version-controlled way
- **Google BigQuery** — as the data warehouse to store and query the hospital data
- **Terraform** — for provisioning and managing Google Cloud resources, so the infrastructure side of things is part of the journey too

## Why hospitals?

With both parents working in the NHS, I've grown up with a real appreciation for how the health system works (and the pressures it faces). Bringing together data on UK hospitals feels like a meaningful first project, something with a personal connection, not just a toy dataset.

This is very much a work in progress and a learning exercise. Feedback and suggestions are always welcome!
